# Scoreboard Tweaks

Improvements and customization for Minecraft's Scoreboard/Sidebar

## Download

Downloads can be found on the [Modrinth page](https://modrinth.com/mod/scoreboardtweaks).

## Features

- Remove Scoreboard
- Remove Scoreboard in Debug HUD
- Remove Score
- Score Color
- Header Color
- Body Color
- Header Shadow
- Body Shadow
- Number Shadow
